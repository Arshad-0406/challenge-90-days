HTML List Elements

HTML provides two main types of lists to organize content:

1. Unordered List (<ul>)
Displays a bulleted list.

Use <li> for list items.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

Example in browser:
• Item 1
• Item 2
• Item 3

2. Ordered List (<ol>)

Displays a numbered list.

Use <li> for list items.

Syntax:
<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ol>

Example in browser:
First item
Second item
Third item

3. Nested Lists

Lists can be nested inside another list to create sub-lists.

Example:
<ul>
  <li>Fruits
    <ul>
      <li>Apple</li>
      <li>Banana</li>
    </ul>
  </li>
  <li>Vegetables
    <ul>
      <li>Carrot</li>
      <li>Spinach</li>
    </ul>
  </li>
</ul>

4. Attributes for Lists

type (only for <ol>) – Changes numbering style:

<ol type="A">
  <li>Option A</li>
  <li>Option B</li>
</ol>

A → Uppercase letters
a → Lowercase letters
I → Uppercase Roman numerals
i → Lowercase Roman numerals

start (for <ol>) – Changes starting number:
<ol start="5">
  <li>Item 5</li>
  <li>Item 6</li>
</ol>

ul styles (using CSS) – Change bullet type: disc, circle, square:

<ul style="list-style-type: square;">
  <li>Square bullet</li>
</ul>

5. Best Practices

Use <ul> for unordered lists and <ol> for ordered lists.

Always wrap list items in <li>.

Use nested lists carefully to maintain readability.

Style lists using CSS instead of adding symbols manually.
