What is <form>?

The <form> tag is used to create an interactive form on a webpage.

Forms allow users to enter data (like text, email, password, file upload, etc.) and send it to a server for processing.

Example: login form, registration form, search box, feedback form.

Syntax
<form action="server_page" method="get|post">
   <!-- Form elements go here -->
</form>

Key Attributes of <form>

action → URL of the server page where data is sent.

<form action="/submit">


method → HTTP method used to send data.

get → Data visible in URL (e.g., search forms).

post → Data sent securely in request body (e.g., login forms).

<form method="post">


target → Where to display response.

_self (default) → same tab.

_blank → new tab.

_parent, _top → for frames.

autocomplete → Suggests saved values (on / off).

novalidate → Skips browser’s automatic validation.

Common Form Elements

Inside <form>, we use form controls:

Input Fields

<input type="text" name="username" placeholder="Enter username">
<input type="password" name="password" placeholder="Enter password">


Radio Buttons & Checkboxes

<input type="radio" name="gender" value="male"> Male
<input type="radio" name="gender" value="female"> Female
<br>
<input type="checkbox" name="subscribe" value="yes"> Subscribe


Dropdown Menu

<select name="country">
  <option value="india">India</option>
  <option value="usa">USA</option>
</select>


Text Area

<textarea name="message" rows="4" cols="30"></textarea>


Buttons

<button type="submit">Submit</button>
<button type="reset">Reset</button>
<button type="button">Click Me</button>

<select> and <option> Tags
What is <select>?

The <select> tag is used to create a drop-down list.

Users can pick one option (default) or multiple options (if allowed).

It is often used in forms for choices like country, gender, city, etc.

What is <option>?

The <option> tag defines individual items inside a <select> dropdown.

Each option can have a value that is sent to the server when selected.

Syntax
<select name="list_name">
  <option value="value1">Text 1</option>
  <option value="value2">Text 2</option>
</select>

Attributes of <select>

name → Name of the dropdown (used when submitting form).

id → For labeling with <label for="">.

multiple → Allows selecting multiple options (Ctrl/Shift + Click).

size → Number of visible options without scrolling.

Attributes of <option>

value → The value sent to server when that option is selected.

selected → Makes an option selected by default.

disabled → Makes an option unselectable.

label → Provides a short label (alternative to text).

Example 1: Basic Dropdown
<form>
  <label for="country">Choose a country:</label>
  <select id="country" name="country">
    <option value="india">India</option>
    <option value="usa">USA</option>
    <option value="uk">UK</option>
  </select>
</form>


If user selects "USA", data sent will be:
country=usa


Example 2: Multiple Selection
<form>
  <label for="skills">Select your skills:</label>
  <select id="skills" name="skills" multiple size="3">
    <option value="html">HTML</option>
    <option value="css">CSS</option>
    <option value="js">JavaScript</option>
  </select>
</form>


Users can select more than one option (e.g., HTML and CSS).

Example 3: Default & Disabled Options
<select name="fruit">
  <option disabled selected>Select a fruit</option>
  <option value="apple">Apple</option>
  <option value="banana">Banana</option>
  <option value="mango">Mango</option>
</select>


First option acts like a placeholder (cannot be selected).
