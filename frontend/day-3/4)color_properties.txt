CSS Color Properties

CSS provides multiple properties to control the color of text, backgrounds, borders, shadows, gradients, etc.

1. color

Defines the text color.

p {
  color: red;      /* named color */
  color: #ff0000; /* hex value */
  color: rgb(255, 0, 0);   /* RGB */
  color: rgba(255, 0, 0, 0.8); /* RGB + alpha */
  color: hsl(0, 100%, 50%);/* HSL */
  color: hsla(0, 100%, 50%, 0.7); /* HSL + alpha */
}

2. background-color

Sets the background color of an element.

div {
  background-color: lightblue;
}

3. border-color

Sets the border color of an element.

Can take 1â€“4 values (top, right, bottom, left).

div {
  border: 2px solid;
  border-color: red green blue yellow;
}

4. outline-color

Defines the color of the outline (extra line around an element).

button {
  outline: 2px solid;
  outline-color: orange;
}

5. text-decoration-color

Sets the color of underline, overline, or line-through.

a {
  text-decoration: underline;
  text-decoration-color: red;
}

6. text-emphasis-color

Used with the text-emphasis property (mainly for East Asian typography).

p {
  text-emphasis: circle;
  text-emphasis-color: green;
}

7. caret-color

Sets the color of the text cursor (caret) in input fields.

input {
  caret-color: red;
}

8. column-rule-color

Used in multi-column layouts.

Defines the color of the line between columns.

div {
  column-count: 3;
  column-rule: 3px solid blue;
  column-rule-color: green;
}

9. text-shadow (color part)

Adds a shadow with a color to text.

h1 {
  text-shadow: 2px 2px 5px gray;
}

10. box-shadow (color part)

Adds a shadow to elements with a defined color.

div {
  box-shadow: 4px 4px 10px rgba(0,0,0,0.5);
}
